events

PARTIAL_MAP

used by the client to request a part of the world


        socket.emit('PARTIAL_MAP', boundaries)

        socket.on('PARTIAL_MAP', map => {
            // use partial map data
        })

MAP

used by the client to request the whole world


        socket.emit('MAP')

        socket.on('MAP', map => {
            // use map data
        })

UPDATED_MAP

used by the server to broadcast changes in the world


        socket.on('MAP_UPDATED', map => {
            // use new map data
        })
