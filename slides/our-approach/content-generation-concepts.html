World generation concepts

Block

  • identifier
  • sides definition (connectors)
  • set of classes
  • set of block based constraints

Connector

  • identifier
  • type
    • blacklist
    • whitelist
  • set of block identifiers
  • set of block classes

World

  • set of blocks
  • set of connectors
  • set of world constraints
  • strategy
  • seed

World instance generation


Iteration 0


    world.start ()

Iteration n + 1


    world.getPartialMap (minX, minY, maxX, maxY)

Constraints


world based

  • initial map size
  • horizontal and vertical boundaries
  • map center

block based

  • blacklist/whitelist
  • minimum distance to other blocks
  • maximum occupation
  • maximum occupation by percentage

Rules

  • blacklist/whitelist
  • maximum occupation
  • maximum occupation by percentage
  • minimum distance to other blocks
  • and more!
